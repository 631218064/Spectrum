## Spectrum 登录页完整样式方案

### 1. 设计概述
登录页延续 Spectrum 的“晨间空气”设计语言，整体风格简洁、通透、浪漫，以浅色渐变背景搭配毛玻璃卡片，品牌Logo采用深色渐变，营造现代、优雅且包容的第一印象。

### 2. 整体布局
- 页面采用**居中单列布局**，主要内容集中在屏幕中央的卡片内。
- 卡片宽度：**400px**（桌面端），**90%**（移动端，最大400px）。
- 卡片与屏幕边缘保持适当间距（上下左右至少24px）。

### 3. 背景
- **背景渐变**：从上到下线性渐变，起始色 `#F5F0E8`（暖米白），结束色 `#E8F0F8`（极浅蓝灰）。

- **装饰元素**：在背景中随机散布极淡的微光粒子（圆形），颜色为 `#A8D5E5`，透明度 0.2，大小 2-4px，缓慢浮动（可选 CSS 动画），增加梦幻感但不抢眼。

### 4. 卡片（Frosted Glass）
- **背景**：半透明白色，`rgba(255, 255, 255, 0.7)`。
- **背模糊**：`backdrop-filter: blur(10px)`，Safari 兼容用 `-webkit-backdrop-filter`。
- **圆角**：`24px`。
- **内边距**：`32px`（桌面端），`24px`（移动端）。
- **边框**：极淡白色边框，`1px solid rgba(255, 255, 255, 0.5)`。
- **阴影**：`0 8px 32px rgba(70, 88, 225, 0.1)`。
- **过渡**：无，保持静态。

### 5. Logo
- **位置**：卡片顶部居中，与卡片上边缘间距 `24px`。
- **字体**：品牌定制字体或使用无衬线粗体（如 Inter Bold），字母间距 `1px`。
- **字号**：`32px`（桌面端），`28px`（移动端）。
- **颜色渐变**：从左到右线性渐变，起始 `#D1599A`（玫粉），结束 `#9D61EC`（紫色）。
  ```css
  background: linear-gradient(90deg, #D1599A, #9D61EC);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
  ```
- **内容**：显示完整品牌名“Spectrum”

### 6. 表单字段
每个字段（邮箱/"Email"、密码/"Password"）包含输入框、占位符。
- **输入框容器**：每个输入框上下间距 `16px`。
- **输入框样式**：
  - 背景：纯白 `#FFFFFF`。
  - 边框：`1px solid #E0E7ED`（极浅灰蓝）。
  - 圆角：`12px`。
  - 内边距：`14px 16px`。
  - 字体：`16px`，颜色 `#2E3B4E`。
  - 占位符：颜色 `#D3D3D3`，字体 `16px`；
  - 聚焦状态：边框变为主色 `#4658E1`，添加 `0 0 0 3px rgba(70, 88, 225, 0.1)` 外发光。

### 7. 登录按钮
- **尺寸**：宽度 100%，高度 `48px`。
- **背景**：主色 `#4658E1`。
- **圆角**：`40px`（全圆）。
- **字体**：白色，`16px`，字重600。
- **边框**：无。
- **阴影**：无（悬浮时增加光晕）。
- **悬浮状态**：
  - 背景色：略深 `#3A4BC7`。
  - 光晕：`0 4px 16px rgba(70, 88, 225, 0.4)`。
  - 轻微放大：`transform: scale(1.02)`。
  - 过渡：`all 0.2s ease`。
- **点击状态**：背景 `#2F44B5`，无缩放。

### 8. 链接
- **注册链接**：位于按钮下方，居中显示。其中“注册”和“Sign up”是可链接样式
  - 文字：`还没有账号？注册`（中）/ `Don't have an account? Sign up`（英）。点击后，跳转至注册表单
  - 字体：`14px`，颜色 `#FF9AA2`（浅粉色）。
  - 悬浮：加下划线。

### 9. 国际化适配
- 所有文本从 `translations.ts` 读取，支持中英文切换。
- 输入框类型：`email` 和 `password`（密码字段需支持显示/隐藏切换，可使用 Ant Design 的 `Input.Password` ）。

### 10. 动效补充
- 按钮悬浮：光晕和放大。
- 背景粒子缓慢浮动（可选，使用 `@keyframes`）。
